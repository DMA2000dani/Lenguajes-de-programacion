Asignatura lenguajes de programación en la que he utilizado un lenguaje de programacion funcional llamado Haskell.

En la carpeta Labs Haskell, se muestran una serie de ejercicios para practicar Haskell.

En la carpeta PrácticaLlull, se puede ver la práctica donde se puede encontrar el enunciado en https://github.com/jordi-petit/lp-llull-2021.
Está práctica esta realizada con ANTLR4 y Python. Consiste en "replicar" un compilador, al interpretar un lenguaje inventado llamado Llull.

En la carpeta PracticaPallus, se puede ver la práctica realizada acorde con el enunciado que se puede ver en https://github.com/jordi-petit/lp-pallus-2021. Esta práctica está realizada con Haskell y consiste a realizar una interpretación lenguajes de programación lógicos como prolog, interpretando ejecuciones utilizando hechos y reglas.	

En la carpeta PracticaPropGraph, se intenta simular un grafo de propiedades, acorde a las especificaciones del enunciado, permitiendo posteriormente leer, mostrar y hacer consultas en el grafo de propiedades. El lenguaje de programación utilizado es Haskell.
